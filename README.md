# Torre de hanoi
Projeto em C com o intuito de elaborar o jogo torre de hanoi
# Objetivo
Desenvolver a experiencia do jogo e seus movimentos em um programa feito em linguagem C
# Como rodar (breve)
# Equipe 
Felipe Gustavo → interface.c | Renderização ASCII das torres no terminal, animações de movimento |

Kauã Alves → pontuacao.c | Cronômetro, cálculo de pontos, highscore, leitura/escrita em arquivo |

Thiago Fontan → automatico.c | Algoritmo recursivo de resolução automática + sistema de dicas |

Caua souza → main.c + menu.c | Menu, telas de início/fim, loop principal do jogo |

Breno cruz → movimentos.c | Validação de jogadas, contador, histórico de movimentos da partida |

Murilo Lima → torres.c | Struct das torres, inicialização, push/pop de discos |
